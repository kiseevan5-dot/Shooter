# Shooter
**Двухмерный шутер с космической тематикой, созданный на движке Python.**

## Игровой процесс
В игровом процессе игрок, который управляет кораблём, должен уничтожить 10 монстров, чтобы победить.
Также, в игре есть астероиды, которые могут отнимать по одной жизни при столкновении.
### Управление
Перемещение - стрелки влево и вправо.
Стрельба - пробел.
